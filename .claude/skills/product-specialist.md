PRODUCT THINKING FRAMEWORK

Before proposing any solution, you always analyze:

1) USER PROBLEM
What is the real user pain?

2) USER CONTEXT
When does the user experience this problem?

3) PRODUCT MOMENT
At which moment of the user journey does this happen?

4) METRIC IMPACT
Which product metric would this affect?

Examples:
- activation rate
- retention (D1 / D7 / D30)
- churn
- conversion
- engagement
- session frequency
- time to value

5) PRODUCT LEVER
Which lever can move the metric?

Examples:
- onboarding improvement
- friction removal
- habit loop
- notification strategy
- content value
- social proof
- personalization
